# 🧑‍🏫 JavaScript 반복문(Loops)

---

## ✅ 1. 반복문이란?

> **반복문(Loop)** 은 특정 조건이 만족될 때까지 **지정된 코드를 반복 실행**하는 제어문입니다.

---

## 📦 2. 자바스크립트의 주요 반복문 종류

| 반복문 종류       | 용도                            |
| ------------ | ----------------------------- |
| `for`        | 반복 횟수가 정해진 반복                 |
| `while`      | 조건을 만족하는 동안 반복                |
| `do...while` | 조건과 관계없이 한 번은 실행되는 반복         |
| `for...in`   | 객체의 키를 반복                     |
| `for...of`   | 반복 가능한 값(iterable)을 반복 (배열 등) |

---

## 🧩 3. `for` 반복문 기본 문법

```javascript
for (초기값; 조건식; 증감식) {
  // 반복 코드
}
```

### ✅ 예제

```javascript
for (let i = 1; i <= 5; i++) {
  console.log("i =", i);
}
```

✅ 출력:

```
i = 1
i = 2
...
i = 5
```

---

## 🔁 4. `while` 반복문

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

* 조건이 **참이면 계속 반복**
* `i++` 등 **종료 조건을 명시하지 않으면 무한 루프**

---

## 🔂 5. `do...while` 반복문

```javascript
let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);
```

* 최소 1회는 실행 보장
* 조건이 false라도 **최초 1회는 실행**

---

## 🔍 6. `for...in` vs `for...of`

| 구문         | 설명                  | 대상                 |
| ---------- | ------------------- | ------------------ |
| `for...in` | 객체의 **key(속성명)** 반복 | 객체, 배열             |
| `for...of` | **값 자체**를 반복        | 배열, 문자열 등 iterable |

---

### ✅ `for...in` 예제

```javascript
const user = { name: "Alice", age: 30 };
for (let key in user) {
  console.log(key + ":", user[key]);
}
```

✅ 출력:

```
name: Alice
age: 30
```

---

### ✅ `for...of` 예제

```javascript
const colors = ["red", "green", "blue"];
for (let color of colors) {
  console.log(color);
}
```

✅ 출력:

```
red
green
blue
```

## ⛔️ 7. 반복문 중단 및 건너뛰기

### `break`

```javascript
for (let i = 0; i < 5; i++) {
  if (i === 3) break;
  console.log(i); // 0,1,2
}
```

### `continue`

```javascript
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i); // 0,1,3,4
}
```

## 📘 8. 반복문 흐름 시각화

### `for` 흐름도:

```text
[초기값] → [조건 검사] ─→ false ─→ 종료
      ↓ true
  [본문 실행]
      ↓
  [증감식] → 다시 조건 검사로
```

## 🔧 10. 반복문 리팩토링 예제

### 기존 코드

```javascript
const fruits = ["apple", "banana", "orange"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
```

### 리팩토링 → `for...of`

```javascript
for (let fruit of fruits) {
  console.log(fruit);
}
```

➡ 코드 가독성, 안정성 향상

---

## 🚀 보너스: 객체 순회 리팩토링

```javascript
const user = { name: "Tom", age: 25 };

// Before
for (let key in user) {
  console.log(key + ":", user[key]);
}

// After: Object.entries
Object.entries(user).forEach(([key, value]) => {
  console.log(`${key}: ${value}`);
});
```

## 🎯 마무리 요약

| 반복문 종류       | 사용 대상       | 특징             |
| ------------ | ----------- | -------------- |
| `for`        | 범위, 횟수 반복   | 가장 기본적인 반복문    |
| `while`      | 조건 반복       | 조건을 만족할 때만 실행  |
| `do...while` | 최소 1회 보장    | 조건이 false여도 실행 |
| `for...in`   | 객체 키 반복     | 객체의 속성 접근      |
| `for...of`   | 배열/문자열 등 반복 | 값 직접 반복        |
