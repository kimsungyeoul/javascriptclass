
# 🧑‍🏫 JavaScript 조건문(Conditional Statements)

---

## ✅ 1. 조건문이란?

> 조건문은 주어진 조건의 **참/거짓 여부에 따라 코드 흐름을 분기**하는 제어문입니다.

---

## 📦 2. 자바스크립트 조건문의 종류

| 구문                      | 용도               |
| ----------------------- | ---------------- |
| `if`, `else if`, `else` | 가장 기본적인 조건 분기    |
| `switch`                | 여러 값에 따라 분기      |
| 삼항 연산자 (`? :`)          | 간단한 조건 분기 (한 줄로) |

---

## 🧩 3. `if` 조건문 기본 문법

```javascript
if (조건) {
  // 조건이 참일 때 실행
} else if (다른조건) {
  // 위 조건이 거짓이고, 이 조건이 참일 때
} else {
  // 모든 조건이 거짓일 때
}
```

---

## ✅ 4. `if` 예제

```javascript
let age = 20;

if (age >= 18) {
  console.log("성인입니다.");
} else {
  console.log("미성년자입니다.");
}
```

---

## 🧮 5. `if` 조건 평가의 진리표

자바스크립트에서는 다음 값이 **false로 간주(falsy)** 됩니다:

```javascript
false, 0, -0, 0n, "", null, undefined, NaN
```

나머지 값은 **true로 간주(truthy)** 됩니다.

```javascript
if ("hello") console.log("true");  // 출력됨
if (0) console.log("false");       // 출력 안 됨
```

---

## 🔁 6. 중첩 if문

```javascript
let score = 85;

if (score >= 90) {
  console.log("A");
} else {
  if (score >= 80) {
    console.log("B");
  } else {
    console.log("C 이하");
  }
}
```

➡ 중첩보다는 `else if`를 추천

---

## 🔀 7. `switch` 조건문

```javascript
let fruit = "apple";

switch (fruit) {
  case "apple":
    console.log("사과입니다.");
    break;
  case "banana":
    console.log("바나나입니다.");
    break;
  default:
    console.log("모르는 과일입니다.");
}
```

### ✅ 특징

* `===`와 같은 **엄격한 비교**를 수행
* `break` 없으면 **다음 case까지 실행됨** (fall-through)

---

## 🧿 8. 삼항 연산자 (`조건 ? 참 : 거짓`)

```javascript
let age = 25;
let message = (age >= 18) ? "성인" : "미성년자";

console.log(message); // "성인"
```

✅ 간단한 조건 분기에 적합
❌ 복잡한 조건에는 사용 지양

---

## ⚠️ 9. 조건문 사용 시 주의사항

| 실수           | 설명                                      |
| ------------ | --------------------------------------- |
| `=`과 `==` 혼동 | `=`는 대입, `==`는 비교                       |
| 중괄호 생략       | if 안의 문장이 2줄 이상일 경우 반드시 `{}` 사용         |
| break 누락     | `switch`에서 break 생략 시 의도치 않게 다음 case 실행 |

## 🧪 10. 실습 문제

### Q1. 점수에 따라 등급을 출력하는 프로그램 작성

```javascript
let score = 72;

// 90점 이상: A
// 80~89: B
// 70~79: C
// 60~69: D
// 그 외: F
```

<details>
<summary>✅ 정답 보기</summary>

```javascript
if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else if (score >= 70) {
  console.log("C");
} else if (score >= 60) {
  console.log("D");
} else {
  console.log("F");
}
```

</details>

### Q2. 사용자가 입력한 요일에 따라 메시지를 출력 (switch 사용)

```javascript
let day = "sunday";

// sunday → "쉬는 날입니다."
// monday → "출근!"
// 그 외 → "일상입니다."
```

<details>
<summary>✅ 정답 보기</summary>

```javascript
switch (day) {
  case "sunday":
    console.log("쉬는 날입니다.");
    break;
  case "monday":
    console.log("출근!");
    break;
  default:
    console.log("일상입니다.");
}
```

</details>

### Q3. 삼항 연산자로 홀수/짝수 구별

## 📊 11. 조건문 흐름도 요약

```text
     조건 A?
      |
     YES → 실행1
      |
    NO
      ↓
   조건 B?
      |
     YES → 실행2
      |
    NO
      ↓
    실행3
```

## 📘 12. 조건문을 대체할 수 있는 문법

| 대체 문법                  | 설명               |    |                      |
| ---------------------- | ---------------- | -- | -------------------- |
| `&&`, \`               |                  | \` | 조건에 따라 실행 또는 디폴트값 제공 |
| `switch`               | 다중 조건 처리에 유리     |    |                      |
| `map`, `object lookup` | 정형화된 값 조건 처리에 유용 |    |                      |

```javascript
let lang = "ko";
let msg = { ko: "안녕", en: "Hello", ja: "こんにちは" }[lang] || "???";
console.log(msg); // "안녕"
```

## ✅ 마무리 요약

| 구문        | 용도         | 특징           |
| --------- | ---------- | ------------ |
| `if`      | 기본 조건문     | 중첩 가능, 블록 필요 |
| `else if` | 다단계 조건문    | 순차 평가        |
| `else`    | 그 외의 모든 경우 | 항상 마지막에 위치   |
| `switch`  | 값 일치 비교    | `break` 필수   |
| 삼항 연산자    | 간단한 조건     | 한 줄 처리 가능    |

## 🧭 보충: 조건문을 평가할 때 주의해야 할 Truthy/Falsy 값

| 값                                              | 평가 결과   |
| ---------------------------------------------- | ------- |
| `false`, `0`, `""`, `null`, `undefined`, `NaN` | ❌ false |
| `"hello"`, `123`, `{}`, `[]`                   | ✅ true  |
