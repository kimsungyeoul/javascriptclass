# 📚 JavaScript 배열(Array)

---

## ✅ 1. 배열이란?

> 배열(Array)은 **여러 개의 값을 순차적으로 저장**할 수 있는 **객체형 자료구조**입니다.
> 자바스크립트에서 배열은 index(인덱스)를 통해 각 요소에 접근할 수 있으며, **0부터 시작**합니다.

```javascript
const fruits = ["사과", "바나나", "오렌지"];
```

* `fruits[0] → "사과"`
* `fruits.length → 3`

## 🧱 2. 배열의 주요 특징

| 특징           | 설명                    |
| ------------ | --------------------- |
| 동적 크기        | 배열 길이는 자동 확장 가능       |
| 다양한 타입 저장 가능 | 숫자, 문자열, 객체 등 혼합 가능   |
| 순서 보장        | 요소는 저장된 순서대로 유지됨      |
| 배열도 객체이다     | typeof 배열은 `"object"` |

## 🛠️ 3. 배열 생성 방법

### 📌 리터럴 방식 (가장 일반적)

```javascript
const arr = [1, 2, 3];
```

### 📌 생성자 방식

```javascript
const arr = new Array(3); // 길이 3인 빈 배열
const arr2 = new Array(1, 2, 3); // 요소 1, 2, 3
```

### 📌 Array.of, Array.from

```javascript
Array.of(7);           // [7]
Array.from("hello");   // ['h','e','l','l','o']
```

---

## 🧭 4. 배열 요소 접근

```javascript
const arr = [10, 20, 30];

console.log(arr[0]); // 10
console.log(arr[arr.length - 1]); // 마지막 요소
```

## 🔄 6. 주요 배열 메서드

| 분류    | 메서드                                         | 설명                   |
| ----- | ------------------------------------------- | -------------------- |
| 탐색    | `includes`, `indexOf`                       | 포함 여부, 위치            |
| 삽입/삭제 | `push`, `pop`, `shift`, `unshift`, `splice` | 끝/앞 추가, 삭제           |
| 변환    | `join`, `reverse`, `sort`                   | 배열 → 문자열, 정렬         |
| 반복    | `forEach`, `map`, `filter`, `reduce`        | 고차 함수 (함수를 파라미터로 받음) |
| 생성    | `slice`, `concat`, `flat`                   | 배열 복사, 병합            |

### 예시

```javascript
let nums = [1, 2, 3];
nums.push(4);      // [1, 2, 3, 4]
nums.pop();        // [1, 2, 3]
```

## 🎯 7. 반복 예제

### `forEach`

```javascript
const nums = [10, 20, 30];
nums.forEach((num, index) => {
  console.log(index + ": " + num);
});
```

### `map` – 새로운 배열 반환

```javascript
const doubled = nums.map(num => num * 2);
// [20, 40, 60]
```

## 🧩 8. 배열 리팩토링 예제

### Before: 반복 코드

```javascript
const prices = [100, 200, 300];
let newPrices = [];
for (let i = 0; i < prices.length; i++) {
  newPrices.push(prices[i] * 1.1);
}
```

### After: `map` 활용

```javascript
const newPrices = prices.map(p => p * 1.1);
```

## 🧠 9. 고차 함수 예제 정리

| 메서드       | 목적              | 반환값    |
| --------- | --------------- | ------ |
| `forEach` | 단순 반복           | 없음     |
| `map`     | 요소 가공 후 새 배열 생성 | 새로운 배열 |
| `filter`  | 조건에 맞는 요소만 필터   | 새로운 배열 |
| `reduce`  | 누적 계산           | 하나의 값  |

```javascript
const nums = [1, 2, 3, 4];
const even = nums.filter(n => n % 2 === 0); // [2, 4]
const sum = nums.reduce((acc, n) => acc + n, 0); // 10
```

## 🐛 10. 디버깅 포인트

* `arr[10]`처럼 존재하지 않는 index 접근 시 `undefined`
* `typeof arr === "object"`임에 유의 → 배열 여부는 `Array.isArray(arr)`로 확인
* `==` 비교 시 참조형 데이터는 같은 참조여야 동일로 인식됨
