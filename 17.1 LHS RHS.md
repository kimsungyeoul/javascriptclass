# 🎯 RHS와 LHS란 무엇인가?

RHS / LHS는 프로그래밍 언어 이론에서 매우 기본적이고 중요한 개념입니다.

* **LHS = Left-Hand Side (좌변)**
* **RHS = Right-Hand Side (우변)**

즉, **대입(할당) 연산자 `=` 기준으로 왼쪽과 오른쪽**을 의미합니다.

```
LHS = RHS
```

이 문장에서:

* `LHS` → 값을 **받는 위치**
* `RHS` → **평가되어 값이 되는 표현식**

---

# 1️⃣ 기본 예제

```js
const x = 10;
```

* LHS: `x`
* RHS: `10`

이 말은:

> “RHS(10)의 값을 LHS(x)에 넣는다.”

---

# 2️⃣ 표현식에서도 RHS/LHS 구분은 동일하다

```js
let result = 5 + 3 * 2;
```

* RHS: `5 + 3 * 2`
* LHS: `result`

---

# 3️⃣ 구조 분해 할당에서 LHS/RHS 차이

이제 중요한 구조 분해 패턴을 보면 LHS와 RHS의 역할이 완전히 다릅니다.

---

## ✔ 배열 구조 분해

```js
const [a, b] = [10, 20];
```

* **LHS:** `[a, b]` → 변수들이 선언되는 위치
* **RHS:** `[10, 20]` → 값이 평가되는 배열

---

## ✔ 객체 구조 분해

```js
const { id: userId } = user;
```

* **LHS:** `{ id: userId }` → 구조 분해 패턴
* **RHS:** `user` → 실제 값이 들어있는 객체

여기서 포인트:

> LHS는 “변수를 선언하는 패턴”
> RHS는 “실제 데이터가 들어 있는 객체”

그래서 LHS에 있는 `{ id: userId }`는 **객체를 만드는 게 아니고**,
“RHS(user)에서 id를 꺼내 userId에 넣는 패턴”일 뿐입니다.

---

# 4️⃣ Spread vs Rest를 이해할 때 LHS/RHS는 더 중요함

## ✔ Spread 연산자 (…)는 RHS에서만 사용된다

Spread는 "펼친다".

```js
const arr2 = [...arr1];
```

여기서 `...arr1`은 **RHS(오른쪽)** 에 있으므로 Spread이다.

## ✔ Rest 파라미터 (…)는 LHS에서만 사용된다

Rest는 "모은다".

```js
const [first, ...rest] = arr;
```

여기서 `...rest`는 **LHS(왼쪽)** 에 있으므로 Rest이다.

→ 바로 이 **LHS/RHS 위치 차이**로 Spread와 Rest를 구분한다.

---

# 5️⃣ 정리 표

| 구분                        | 의미                        |
| ------------------------- | ------------------------- |
| **LHS (Left-Hand Side)**  | “값을 받는 쪽” – 변수 선언 위치      |
| **RHS (Right-Hand Side)** | “값이 평가되는 표현식” – 데이터가 있는 곳 |

구조 분해 할당에서:

| 위치      | 의미                                        |
| ------- | ----------------------------------------- |
| **LHS** | 구조 분해 패턴 (`{ id }`, `[a]`, `{ ...rest }`) |
| **RHS** | 실제 데이터 (`user`, `[1,2,3]`, 함수 호출 결과 등)    |

Spread vs Rest:

| 구분     | 쓰임 | 위치         |
| ------ | -- | ---------- |
| Spread | 펼침 | **RHS에서만** |
| Rest   | 모음 | **LHS에서만** |

---

# 6️⃣ 완전 쉽게 요약

* `LHS = RHS`

* LHS는 "여기에 값을 넣어라"

* RHS는 "이 값을 평가해서 왼쪽에 넣어라"

* Spread: RHS에서 쓰면 "펼쳐라"

* Rest: LHS에서 쓰면 "모아라"
