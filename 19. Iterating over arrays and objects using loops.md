# 📘 자바스크립트 반복문으로 배열과 객체 순회하기

---

## 🧩 1. 배열 순회 방법

### 🔸 1.1 `for` 루프

```javascript
const arr = ['apple', 'banana', 'cherry'];

for (let i = 0; i < arr.length; i++) {
  console.log(`${i}: ${arr[i]}`);
}
```

> 가장 전통적인 방식. 인덱스를 활용해 배열 요소에 접근 가능

---

### 🔸 1.2 `for...of` 루프

```javascript
for (const fruit of arr) {
  console.log(fruit);
}
```

> ES6 이후 사용. **배열, 문자열, Set, Map** 등 iterable 객체를 순회할 수 있음.

---

### 🔸 1.3 `forEach()` 메서드

```javascript
arr.forEach((item, index) => {
  console.log(`${index}: ${item}`);
});
```

> 함수형 스타일. 내부에서 콜백 함수 사용.

---

### 🔸 1.4 `map()` 메서드 (새 배열 생성)

```javascript
const upper = arr.map(fruit => fruit.toUpperCase());
console.log(upper); // ['APPLE', 'BANANA', 'CHERRY']
```

> `forEach`와 달리 **리턴값으로 새 배열 반환**

---

### 🔸 1.5 `for...in` 루프 (주의)

```javascript
for (let key in arr) {
  console.log(key, arr[key]);
}
```

> 배열의 \*\*인덱스(문자열)\*\*를 순회. 배열보다는 객체에 사용 권장.

---

## 🧪 배열 실습 예제

```javascript
const scores = [90, 75, 88];

console.log("▶ forEach:");
scores.forEach((s, i) => console.log(`${i}: ${s}`));

console.log("▶ map:");
const doubled = scores.map(s => s * 2);
console.log(doubled);

console.log("▶ for...of:");
for (let score of scores) {
  console.log(score);
}
```

## 🧩 2. 객체 순회 방법

### 🔸 2.1 `for...in` 루프

```javascript
const user = {
  name: "Alice",
  age: 30,
  job: "Developer"
};

for (let key in user) {
  console.log(`${key}: ${user[key]}`);
}
```

> 객체의 \*\*모든 열거 가능한 속성(key)\*\*을 순회함.

---

### 🔸 2.2 `Object.keys()` + `forEach()`

```javascript
Object.keys(user).forEach(key => {
  console.log(`${key}: ${user[key]}`);
});
```

> `Object.keys()`는 **열거 가능한 key 배열** 반환

---

### 🔸 2.3 `Object.entries()`

```javascript
for (let [key, value] of Object.entries(user)) {
  console.log(`${key} = ${value}`);
}
```

> 객체를 `[key, value]` 형태의 배열로 변환하여 `for...of`로 순회

---

### 🔸 2.4 `Object.values()`

```javascript
console.log(Object.values(user)); // ["Alice", 30, "Developer"]
```

> 객체의 **값(value)들만 배열로 추출**

---

## 🧪 객체 실습 예제

```javascript
const book = {
  title: "JavaScript Deep Dive",
  pages: 450,
  author: "이웅모"
};

console.log("▶ for...in:");
for (let key in book) {
  console.log(`${key}: ${book[key]}`);
}

console.log("▶ Object.entries:");
Object.entries(book).forEach(([k, v]) => {
  console.log(`${k} = ${v}`);
});
```

---

## ⚠️ 정리: 배열 vs 객체 순회 방식 비교

| 반복 방식              | 배열 ✅ | 객체 ✅ | 설명                       |
| ------------------ | ---- | ---- | ------------------------ |
| `for`              | ✅    | ❌    | 배열 전통 방식                 |
| `forEach()`        | ✅    | ❌    | 배열 전용, 함수형 스타일           |
| `for...of`         | ✅    | ❌    | iterable 대상 순회           |
| `for...in`         | ⚠️   | ✅    | 객체 속성 순회 (배열엔 비추)        |
| `Object.keys()`    | ✅    | ✅    | key 배열 반환 후 순회           |
| `Object.values()`  | ✅    | ✅    | value 배열 반환              |
| `Object.entries()` | ✅    | ✅    | `[key, value]` 배열로 순회 가능 |

---

## 🧠 보너스: 순회 리팩토링 패턴

### 🔧 잘못된 for...in 사용 (배열에 비추천)

```javascript
const arr = ['a', 'b', 'c'];
for (let i in arr) {
  console.log(arr[i]); // 가능은 하지만 비추천
}
```

### ✅ 개선

```javascript
for (const item of arr) {
  console.log(item); // 더 안전하고 깔끔함
}
```
